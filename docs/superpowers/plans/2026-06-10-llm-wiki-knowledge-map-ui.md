# llm-wiki Knowledge Map UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a static knowledge-map homepage and improve shared reader components for llm-wiki without changing the Markdown knowledge model.

**Architecture:** Extend the existing single-file HTML builder with explicit page metadata, topic grouping, recent-update parsing, and smaller render functions. Keep all runtime behavior static and client-side: theme toggle, search filtering, and TOC active state only. Style the new homepage and reader components through the existing shared CSS file.

**Tech Stack:** Python 3.11 script via `uv`, `markdown>=3.6`, static HTML, vanilla JavaScript, CSS custom properties.

---

- Modify `wiki/html-assets/build.py`: add metadata collection, JSON-safe search index generation, topic groups, recent updates, homepage render helpers, and revised page shell markup.
- Modify `wiki/html-assets/style.css`: add homepage modules, reader header, related panel, accessible focus states, responsive behavior, and replace callout side stripes.

- Regenerate generated pages under `wiki/html/{entities,concepts,sources,analysis}/` and `wiki/html/log.html`: shared reader shell/style changes generated by `build.py`.
- Do not modify `raw/`, `wiki/*.md`, `AGENTS.md`, or `README.md`.

```python
INDEX_LINK_DESC_RE = re.compile(
    r'<a class="wikilink" href="(?P<href>[^"]+)">(?P<label>[^<]+)</a>\s*—\s*(?P<desc>[^<\n]+)'
)


def extract_index_descriptions(index_html: str) -> dict[str, str]:
    descriptions: dict[str, str] = {}
    for m in INDEX_LINK_DESC_RE.finditer(index_html):
        href = htmllib.unescape(m.group("href"))
        desc = htmllib.unescape(m.group("desc")).strip()
        descriptions[href] = desc
    return descriptions
```

- [ ] **Step 2: Add topic grouping helper**

Add this before `build_site_index()`:

```python
def build_topic_groups(pages: list[PageMeta]) -> list[dict[str, object]]:
    by_stem = {p.stem: p for p in pages}
    groups: list[dict[str, object]] = []
    for cfg in TOPIC_GROUPS:
        tagset = cfg["tags"]
        preferred_stems = cfg["preferred"]
        matches = [
            p for p in pages
            if tagset.intersection(set(p.tags))
            or any(token in p.title.lower() for token in tagset)
            or p.stem in preferred_stems
        ]
        unique: dict[str, PageMeta] = {p.href: p for p in matches}
        ordered = [by_stem[s] for s in preferred_stems if s in by_stem]
        ordered.extend(
            sorted(
                (p for p in unique.values() if p.stem not in preferred_stems),
                key=lambda p: (p.category != "analysis", p.title.lower()),
            )
        )
        groups.append(
            {
                "title": cfg["title"],
                "description": cfg["description"],
                "count": len(unique),
                "links": ordered[:4],
            }
        )
    return groups
```

- [ ] **Step 3: Add recent log parser**

Add this before `build_site_index()`:

```python
LOG_ENTRY_RE = re.compile(r"^## \[(?P<date>\d{4}-\d{2}-\d{2})\]\s+(?P<kind>[^|]+)\|\s*(?P<title>.+)$")


def parse_recent_updates(limit: int = 8) -> list[dict[str, str]]:
    log_path = WIKI / "log.md"
    if not log_path.exists():
        return []
    updates: list[dict[str, str]] = []
    for line in log_path.read_text(encoding="utf-8").splitlines():
        m = LOG_ENTRY_RE.match(line.strip())
        if not m:
            continue
        updates.append(
            {
                "date": m.group("date"),
                "kind": m.group("kind").strip(),
                "title": m.group("title").strip(),
                "href": "log.html",
            }
        )
    return updates[:limit]
```

- [ ] **Step 5: Replace homepage search JavaScript**

Use this script inside `extra_body` after the modules:

```html
<script>
  const SEARCH_INDEX = {search_json};
  const input = document.getElementById("search-input");
  const out = document.getElementById("search-results");
  function search(q) {{
    q = q.trim().toLowerCase();
    if (!q) {{ out.innerHTML = ""; return; }}
    const hits = SEARCH_INDEX.filter(e => {{
      const tags = (e.tags || []).join(" ");
      return [e.title, e.category, e.categoryLabel, tags, e.description]
        .join(" ")
        .toLowerCase()
        .includes(q);
    }}).slice(0, 30);
    out.innerHTML = hits.map(e =>
      `<li><a class="wikilink" href="${{e.href}}">${{e.title}}</a>` +
      `<small>${{e.categoryLabel}} · ${(e.tags || []).slice(0, 4).join(", ")}</small>` +
      `${{e.description ? `<p>${{e.description}}</p>` : ""}}</li>`
    ).join("");
  }}
  input.addEventListener("input", e => search(e.target.value));
</script>
```
